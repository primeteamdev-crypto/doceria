# 🎂 Açúcar e Afeto | Bolos Personalizados Premium

![Açúcar e Afeto](public/og-image.jpg)

**Açúcar e Afeto** é uma landing page premium desenvolvida para uma confeitaria artesanal localizada na Barra da Tijuca, Rio de Janeiro. O site foca em proporcionar uma experiência elegante e intuitiva para clientes que buscam bolos de festa personalizados e acabamento de confeitaria fina.

## ✨ Destaques

- **Design Premium**: Visual sofisticado utilizando tipografia clássica (Playfair Display) e moderna (Raleway).
- **Quiz de Personalização**: Um "Cake Quiz" interativo para ajudar o cliente a escolher o bolo ideal.
- **Portfólio Dinâmico**: Carrosséis interativos exibindo os trabalhos realizados.
- **Prova Social**: Seção de depoimentos e estatísticas de impacto.
- **Otimização SEO**: Configurado para buscas locais no Rio de Janeiro (Barra da Tijuca).
- **Totalmente Responsivo**: Experiência fluida em dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com as melhores ferramentas do ecossistema moderno:

- **React + Vite**: Para um desenvolvimento rápido e performance otimizada.
- **TypeScript**: Garantindo maior segurança e manutenibilidade do código.
- **Tailwind CSS**: Estilização moderna e responsiva.
- **Shadcn/UI**: Componentes de interface acessíveis e elegantes.
- **Radix UI**: Primitivos de componentes para acessibilidade.
- **TanStack Query**: Gerenciamento eficiente de estados e cache.
- **Lucide React**: Biblioteca de ícones moderna.

## 🚀 Começando

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado.
- Gerenciador de pacotes `npm` (ou `bun`, `pnpm`).

### Instalação e Execução

1. **Clone o repositório:**

   ```sh
   git clone <URL_DO_REPOSITORIO>
   cd acucar-afeto
   ```

2. **Instale as dependências:**

   ```sh
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**

   ```sh
   npm run dev
   ```

O site estará disponível em `http://localhost:5173`.
